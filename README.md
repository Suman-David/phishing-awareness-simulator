Advanced Phishing Awareness Simulation Tool

Overview
An educational simulator built with Python + Flask + SQLAlchemy to raise awareness about phishing attacks.  
Users can safely experience simulated phishing scenarios and learn how to identify suspicious emails and links.

Features
- Interactive phishing simulations
- Persistent logging of user responses
- Dashboard with visual analytics
- Ethical, safe, and demo-ready design

Tech Stack
- Python
- Flask
- SQLAlchemy
- HTML/CSS

Purpose
Created as a portfolio project to demonstrate cybersecurity awareness and ethical simulation design.  
Intended for educational use only.

Author
